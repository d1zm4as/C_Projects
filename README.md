# C_Projects

Collection of small C experiments focused on low-level programming fundamentals.

## Projects

- `Build System`: Bash-based scaffolding tool to create C projects from templates.
- `Analog Clock`: Raylib-based analog clock that renders hands from local system time.
- `FibBenchmark`: CLI to benchmark big-Fibonacci implementations with a reference fast-doubling check.
- `QrCodeGenerator`: Small QR code generator library (C source + header, still working).
- `Simple Password Generator`: Generates a random password from a fixed character set.
- `Ray Tracing`: SDL2-based 2D raycasting demo with interactive mouse-controlled light source and occluder.
- `TCP Server`: Minimal TCP server that listens on port `8181` and replies to one client.
- `TCP Sockets`: Basic TCP client that performs a simple HTTP `HEAD` request.
- `Toralizer`: LD_PRELOAD wrapper that routes `connect()` calls through a SOCKS4 proxy (Tor-style).
- `UI`: Terminal UI experiments using `ncurses`.
- `Xor_encryption`: Small XOR demonstration for reversible byte-level encryption.
- `communication_protocols`: Protocol notes and C examples (UART, SPI, I2C, TCP/UDP, CAN, Modbus RTU, MQTT, Zigbee).
- `tic-tac-toe_C`: Console Tic-Tac-Toe with PvP and PvM modes.
- `zipador`: Small archiving/compression experiments, still working on it.

## Notes

Each folder is independent and can be compiled/run on its own.
