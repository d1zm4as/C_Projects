# C_Projects

Collection of small C experiments focused on low-level programming fundamentals.

## Projects

- `TCP Server`: Minimal TCP server that listens on port `8181` and replies to one client.
- `TCP Sockets`: Basic TCP client that performs a simple HTTP `HEAD` request.
- `UI`: Terminal UI experiments using `ncurses`.
- `Xor_encryption`: Small XOR demonstration for reversible byte-level encryption.

## Notes

Each folder is independent and can be compiled/run on its own.
